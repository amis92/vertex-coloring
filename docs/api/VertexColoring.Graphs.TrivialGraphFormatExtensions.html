WriteColoringAsTgf(TextWriter, GraphColoring)

Saves given coloring into writer in TGF-like text representation. The color of each vertex is saved as a second number, after vertex id. So a vertex with id=1, label=sth colored with number 132 will be saved in TGF vertex list as "1 132 sth".
